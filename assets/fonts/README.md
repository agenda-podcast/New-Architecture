# Typography Fonts

Fonts for titles, lower thirds, and text overlays.

## Format

- TrueType (.ttf) or OpenType (.otf)
- Full character set (A-Z, 0-9, symbols)
- Web font compatible

## Required Fonts

- [ ] **Title font**: Bold, impactful (e.g., Montserrat Bold)
- [ ] **Body font**: Clean, readable (e.g., Inter Regular)

## Sources

- **Google Fonts**: https://fonts.google.com/
- **Font Squirrel**: https://www.fontsquirrel.com/ (100% free section)
- **League of Moveable Type**: https://www.theleagueofmoveabletype.com/

## Recommendations

### Title Fonts
- **Montserrat**: Bold, modern sans-serif
- **Bebas Neue**: Tall, condensed display font
- **Oswald**: Strong, narrow sans-serif

### Body Fonts
- **Inter**: Highly readable UI font
- **Roboto**: Google's versatile sans-serif
- **Open Sans**: Friendly, neutral sans-serif

## Usage

In Blender:
1. Load external font (Font menu)
2. Assign to Text object or VSE text strip
3. Adjust size, spacing, tracking
