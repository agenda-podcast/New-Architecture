## Attribution Requirements

Some assets require attribution. When attribution is required:

1. Credit is included in `ATTRIBUTION.txt` in asset directory
2. Credit is included in video description (if applicable)
3. Credit format specified by license

## Prohibited Licenses

The following licenses are NOT used in this project:

- **All Rights Reserved**: Not free/open
- **Non-Commercial Only**: Conflicts with MIT license
- **Share-Alike (CC-BY-SA)**: Viral license, conflicts with MIT
- **GPL**: Viral license, conflicts with MIT
- **Unknown/Unclear**: Cannot verify freedom to use

## Verification Process

Before adding any asset:

1. **Find license page** on source website
2. **Verify commercial use** is explicitly allowed
3. **Check for restrictions** (attribution, share-alike, etc.)
4. **Download license file** if provided
5. **Document in LICENSE.txt** in asset directory
6. **Note attribution** in ATTRIBUTION.txt if required

## Legal Compliance

This project takes licensing seriously:

1. **All assets are documented**: License + attribution
2. **No gray-area sources**: Only clear, explicit licenses
3. **Commercial use verified**: Essential for MIT compatibility
4. **No viral licenses**: Avoids GPL, CC-BY-SA
5. **Attribution honored**: Credits provided where required

## Questions?

If unsure about an asset's license:
- **Don't add it** until verified
- **Contact the creator** for clarification
- **Find an alternative** with clear licensing
- **Create your own** asset instead
